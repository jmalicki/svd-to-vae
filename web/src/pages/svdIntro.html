Reflections, zeros, and stretches

Source on GitHub

Often you want a vector — or a matrix column — lined up with a coordinate axis: one nonzero entry, the rest zero, length unchanged. A reflection can do that in one step. This page builds that operator from scratch, then uses it to clear zeros under a pivot and read the stretches that remain.

Building a reflection

Chapter 1 used rotations: multiply by an orthogonal matrix with determinant $+1$, and every length stays the same. A reflection is the other standard length-preserving move. Pick a line through the origin — the mirror — and send every point to its twin on the other side. Lengths are unchanged, but the plane flips (determinant $-1$).

A mirror through the origin is completely described by a unit vector $n$ at right angles to it (the normal). The sidebar keeps the mirror and a probe vector $x$ in reach while you scroll the construction; every figure uses the same choices.

1 · The mirror and its normal

Orange is the mirror; green is a unit normal $n$ (so $\|n\| = 1$ and $n$ is perpendicular to the mirror). You only need one of these: the line determines $n$ up to sign, and $n$ determines the line. Replacing $n$ by $-n$ describes the same reflection.

Orange = mirror. Green = $n$. Gray = coordinate axes.

2 · Split $x$ into parallel and normal pieces

Fix a probe vector $x$ (blue). Project it onto the normal direction: the scalar $n^{\top}x$ is how far $x$ sits on the $n$-side of the mirror, and the vector

$$x_{n} = (n^{\top}x)\,n$$

is that piece of $x$ (sky-blue dashed in the figure). What remains, $x_{\parallel} = x - x_{n}$, lies in the mirror (purple). So $x = x_{\parallel} + x_{n}$: a part parallel to the mirror and a part along $n$.

Blue = $x$. Purple = $x_{\parallel}$. Sky dashed = $x_n$ (construction piece along $n$).

3 · Flip the normal piece

To reflect, keep the part in the mirror and reverse the part along $n$: $Hx = x_{\parallel} - x_{n}$. Since $x_{\parallel} = x - x_{n}$, that is the same as subtracting the normal piece twice:

$$Hx = x - 2(n^{\top}x)\,n.$$

In the figure, orange is $Hx$. The dashed segment is the jump across the mirror; $x$ and $Hx$ are the same distance from it, on opposite sides. In particular $\|Hx\| = \|x\|$.

Blue = $x$. Orange = $Hx$. Dashed = the flip across the mirror.

4 · Write it as a matrix

The rule $x \mapsto x - 2(n^{\top}x)\,n$ is linear in $x$. Factoring out $x$ gives the Householder matrix

$$H = I - 2nn^{\top}.$$

Reflecting twice undoes itself, so $H^{2} = I$. Lengths (and angles, up to orientation) are preserved, so $H$ is orthogonal: $H^{\top}H = I$. The orientation flip is $\det H = -1$, whereas chapter 1’s rotations had $\det = +1$. From here on, “apply a reflection” means multiply by a matrix of this form.

Check: find $n$ for a prescribed image

Above you turned the mirror and watched $Hx$ move — given $n$, compute the image. The useful direction in algorithms is the other way: fix a vector $x$ and a same-length target $y$, and find the $n$ (equivalently the mirror) such that $Hx = y$. That inverse question is what Householder reflections are for; the next section uses it to aim onto an axis.

Why aim onto an axis?

Same inverse skill as above, with a target that pays rent later. Suppose you have a vector $a$ pointing every which way, and you wish it lined up with the first coordinate axis — all other coordinates zero — without changing its length. Then later algebra only has to care about one number ($\|a\|$) instead of a whole list of coupled entries. Pick the special target $t = (\|a\|, 0)$ and find $n$ so $Ha = t$ — exactly “make $Hx$ hit $y$,” with $y = t$ on the axis. People call that choice a Householder reflection; the point is the goal, not the name.

Use the sidebar: set $a$, then turn the mirror until $Ha$ has height $0$. Afterward we show the formula that builds that $n$ from $a$ and $t$ automatically.

Construction of that $n$

Get $|(Ha)_2| = 0$ above; then we show the formula for $n$.

If the target is $t = (\|a\|, 0)$, the normal points along $a - t$ (perpendicular bisector of the segment from $a$ to $t$). Normalize for unit $n$, then $H = I - 2nn^{\top}$ sends $a$ to $t$.

$$n = \frac{a - t}{\|a - t\|},\qquad t = (\|a\|, 0)$$

Same idea on a matrix column

The previous section aimed a 2D arrow $a$ onto the $x$-axis: we chose the mirror so $Ha = (\|a\|, 0)^{\top}$ — one nonzero entry, length unchanged. A matrix column is the same object in higher dimension. Write $\mathrm{col}_1$ for the first column of $A$. Build the Householder $H$ that sends that column to the first coordinate axis:

$$H\,\mathrm{col}_1 = \bigl(\|\mathrm{col}_1\|,\,0,\,\ldots,\,0\bigr)^{\top}.$$

Entries $2$ through $n$ are zero because we aimed there — not because a random reflection would wipe them. The mirror is chosen (via $n \propto \mathrm{col}_1 - t$ with $t = (\|\mathrm{col}_1\|, 0, \ldots, 0)^{\top}$) so the image of $\mathrm{col}_1$ is exactly that axis vector. Then replace $A$ by $HA$: every column is reflected by the same $H$, so the rest of the matrix moves too, but column 1 is now a single pivot with zeros below.

Simplify from both sides

Zeros can come from the left and the right. A left Householder aims column 1 onto the axis. Then a right rotation zeros the remaining off-diagonal in $2\times 2$. What is left on the diagonal are the stretch amounts — same axis lengths as on the ellipse. For larger matrices the same idea (orthogonal maps from both sides) is how one thins $A$ down; the next page names the finished factorization.

In short

Reflections preserve lengths like rotations. A Householder chooses a mirror so a column lands on an axis — that introduces zeros. Left and right orthogonal maps can thin a matrix until the leftover diagonal holds the stretch amounts.
